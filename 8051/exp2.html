Write an 8051 ASM program to move a block of five data from 50H RAM to External memory 4000H, add the same and store the result in 5000H (High Byte) and 5001H(Lower Byte).

MOV DPTR, #50H ; Initialize DPTR with source address
MOVX A, @DPTR ; Move data from source to accumulator
MOV R0, A ; Move data from accumulator to register R0
INC DPTR ; Increment DPTR to point to next location
MOVX A, @DPTR ; Move data from source to accumulator
ADD A, R0 ; Add data in accumulator with data in register R0
MOV R1, A ; Move result to register R1
INC DPTR ; Increment DPTR to point to next location
MOVX A, @DPTR ; Move data from source to accumulator
ADD A, R1 ; Add data in accumulator with data in register R1
MOV R2, A ; Move result to register R2
INC DPTR ; Increment DPTR to point to next location
MOVX A, @DPTR ; Move data from source to accumulator
ADD A, R2 ; Add data in accumulator with data in register R2
MOV R3, A ; Move result to register R3
INC DPTR ; Increment DPTR to point to next location
MOVX A, @DPTR ; Move data from source to accumulator
ADD A, R3 ; Add data in accumulator with data in register R3
MOV R4, A ; Move result to register R4
; Store the result in memory locations 5000H and 5001H
MOV A, R4 ; Move the contents of register R4 into accumulator
MOVX @DPTR, A ; Store the contents of accumulator at memory location 5000H (High Byte)
INC DPTR ; Increment DPTR to point to next location
MOV A, R3 ; Move the contents of register R3 into accumulator
MOVX @DPTR, A ; Store the contents of accumulator at memory location 5001H (Lower Byte)
